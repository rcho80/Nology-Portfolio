Collection of Projects

Feature Project:    Mothdx (Client Project)

Mothdx produces nucleic-acid-based diagnostic devices for rapid and accurate testing. Our group task was to build an MVC web app with React, a REST API backend with Spring boot and MySQL for data management.
Key learnings were API integration, GIT flow, agile practices, data management & meeting client MVP requirements & timelines.

Feature Project:    Morse Code Translator

Morse code translator built using vanilla JavaScript, SASS & HTML.
Key learnings were DOM manipulation, BEM for styling structure and handling outputs of JS functions.

Feature Project:    Ecommerce Site

Ecommerce site built using JavaScript & REACT framework with persistent data management via Firebase.
Key learnings were building functional REACT components using hooks and incorporating CRUD operations for persistent cart management.

Feature Project:    Online Library

Online digital library built using JavaScript & SASS along with API integration with Google Books.
Key learnings were researching and using API docs to build solutions, API integration and returning backend data seamlessly to the frontend.

Feature Project:   Digital Calculator

Functional calculator built with vanilla JavaScript, SASS and HTML.
Key learnings were writing JavaScript to handle & display multiple user inputs & functions and designing an appealing product for the end user.
